```markdown
## Unicode直接引用
📊 = U+1F4CA
🚀 = U+1F680
🎯 = U+1F3AF

# 使用短代码（会自动转换）
:bar_chart: 会显示为 📊
:rocket: 会显示为 🚀
:hammer_and_wrench: 会显示为 🛠️
:dart: 会显示为 🎯
:computer: 会显示为 💻
:books: 会显示为 📚
```

## 参考表

| Emoji | 短代码                       | 常见用途         |
| :---- | :--------------------------- | :--------------- |
| 📊     | `:bar_chart:`                | 数据、统计、图表 |
| 🚀     | `:rocket:`                   | 启动、部署、快速 |
| 🎯     | `:dart:`                     | 目标、焦点、精准 |
| 🛠️     | `:hammer_and_wrench:`        | 工具、配置、设置 |
| 🔧     | `:wrench:`                   | 工具、修复       |
| ⚡     | `:zap:`                      | 快速、性能、闪电 |
| 📈     | `:chart_with_upwards_trend:` | 增长、趋势       |
| 📦     | `:package:`                  | 包、模块、发布   |
| 🔗     | `:link:`                     | 链接、连接       |
| 💻     | `:computer:`                 | 编程、计算机     |
